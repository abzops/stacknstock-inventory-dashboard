Supabase Setup + Authentication

This dashboard can run locally, but database mode requires Supabase URL, anon/publishable key, and Supabase Auth.

window.SNS_SUPABASE_URL = "https://YOUR_PROJECT.supabase.co";
window.SNS_SUPABASE_ANON_KEY = "YOUR_ANON_OR_PUBLISHABLE_KEY";

Run supabase-auth-policies.sql after creating users to restrict table access to signed-in users.
